# Usage
<!-- usage -->
```sh-session
$ npm install -g torrent-cli
$ torrent-cli COMMAND
running command...
$ torrent-cli (-v|--version|version)
torrent-cli/0.0.1 linux-x64 node-v11.15.0
$ torrent-cli --help [COMMAND]
USAGE
  $ torrent-cli COMMAND
...
```
